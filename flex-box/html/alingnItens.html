  • align-items: Controla o alinhamento vertical dos itens dentro do contêiner ao longo do eixo transversal (perpendicular ao flex-direction). Se o alinhamento for em linha o centro do axis é na vertical ja em coluna o centro do axis é horizontal

    • stretch (padrão): Estica os itens para preencher o contêiner.
    • flex-start: Alinha os itens no início do eixo cruzado (topo para eixo vertical).
    • flex-end: Alinha os itens no final do eixo cruzado (base para eixo vertical).
    • center: Alinha os itens no centro do eixo cruzado.
    • baseline: Alinha os itens com base na linha base do texto.

    • fluxo row
