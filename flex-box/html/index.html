Tipos de displays

Display block

Por padrão, elementos como (div,section,p,h1,h2,ul,li,article) são display block e elementos como (a,span,b,i) são display inline Um elemento com display: block ocupa toda a largura disponível. Ele cria uma "quebra de linha" antes e depois dele, o que significa que nenhum outro elemento pode ficar ao lado dele.

Display flex

Define um contêiner flexível que organiza seus itens internos de maneira eficiente. Ele permite distribuir o espaço entre itens e alinhar conteúdo de várias maneiras usando propriedades adicionais como justify-content, align-items, e flex-direction.
