order:

Define a ordem em que os itens flex são exibidos dentro do contêiner flex. O valor padrão é 0, mas você pode usar valores positivos ou negativos para reordenar os itens sem alterar o HTML.

order: fluxo row

O eixo principal é horizontal, e vamos usar a propriedade order para modificar a sequência de exibição dos itens. Como Funciona: order-3: O item será exibido na terceira posição. order-1: O item será exibido na primeira posição. order-4: O item será exibido na quarta posição. order-2: O item será exibido na segunda posição.

Box 1 (order: 3)
Box 2 (order: 1)
Box 3 (order: 4)
Box 4 (order: 2)

order: fluxo column

O eixo principal é vrtical, e vamos usar a propriedade order para modificar a sequência de exibição dos itens. Como Funciona: order-3: O item será exibido na terceira posição. order-1: O item será exibido na primeira posição. order-4: O item será exibido na quarta posição. order-2: O item será exibido na segunda posição.

Box 1 (order: 3)
Box 2 (order: 1)
Box 3 (order: 4)
Box 4 (order: 2)

Explicação: A propriedade order altera a ordem de exibição dos itens no contêiner flex. O valor padrão é 0, mas você pode atribuir valores positivos ou negativos para reordenar os itens.
Ao usar order, você não precisa alterar a estrutura HTML. A ordem dos itens é controlada apenas via CSS.
Esta propriedade funciona tanto para layouts horizontais (row) quanto para layouts verticais (column), oferecendo flexibilidade no posicionamento dos elementos sem modificar a marcação.
