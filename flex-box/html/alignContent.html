  • align-content: Controla o espaçamento entre as linhas de itens quando há quebra de linha.
    • stretch (padrão): Estica as linhas para preencher o contêiner.
    • flex-start: Alinha as linhas no início do contêiner.
    • flex-end: Alinha as linhas no final do contêiner.
    • center: Centraliza as linhas no contêiner.
    • space-between: Distribui as linhas com espaços uniformes entre elas.
    • space-around: Distribui as linhas com espaços uniformes ao redor delas.

    • Align content fluxo row

    Vamos usar align-content para controlar o espaçamento entre as linhas em um contêiner com várias quebras de linha.
