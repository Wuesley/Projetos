flex-basis

Define o tamanho inicial de um item flex antes de o espaço ser distribuído de acordo com as propriedades flex-grow e flex-shrink. O flex-basis pode ser considerado o "tamanho preferido" de um item flex.

flex-basis fluxo row

Como Funciona:
basis-100: O item inicia com um tamanho de 100px.
basis-150: O item inicia com um tamanho de 150px.
basis-200: O item inicia com um tamanho de 200px.
basis-auto: O item inicia com o tamanho automático baseado no conteúdo interno ou no tamanho definido pelas regras CSS normais.

Box 1 (flex-basis: 100px)
Box 2 (flex-basis: 150px)
Box 3 (flex-basis: 200px)
Box 4 (flex-basis: auto)

flex-basis fluxo column

Como Funciona:
basis-100: O item começa com uma altura de 100px.
basis-150: O item começa com uma altura de 150px.
basis-200: O item começa com uma altura de 200px.
basis-auto: O item usa o tamanho baseado no conteúdo ou nas regras de layout normais.

Box 1 (flex-basis: 100px)
Box 2 (flex-basis: 150px)
Box 3 (flex-basis: 200px)
Box 4 (flex-basis: auto)

Explicação:
A propriedade flex-basis especifica o tamanho inicial de um item flex antes de o espaço ser distribuído com base nas propriedades flex-grow e flex-shrink.
O valor auto para flex-basis usa o tamanho calculado a partir das regras CSS padrão ou do conteúdo do item.
flex-basis é especialmente útil para criar layouts flexíveis onde você deseja controlar o tamanho inicial dos itens sem afetar o comportamento de crescimento ou encolhimento.
