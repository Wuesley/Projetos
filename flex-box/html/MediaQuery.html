Media Query é uma técnica que permite aplicar estilos específicos a diferentes tamanhos de tela, resoluções, ou outros aspectos do ambiente em que a página web está sendo visualizada. Ela é fundamental para criar layouts responsivos, ou seja, páginas que se ajustam a diferentes dispositivos, como desktops, tablets e smartphones.

Primeiro ponto em que devemos nos atentar é na configuração da meta tag, basta digitar meta: vp ela possibilita que o nosso site seja melhor visto em varios outros dispositivos que se deseja ser adaptado.

temos 3 modos:

  • screen - quando quero apenas modificar a tela;
  • print - quando o usuário quer imprimir a tela;
  • speech - reconhecimento por voz do que ta na pagina;
  • all - Abrange todas as resoluções. Normalmente usamos ele, ao inves de usar individualmente

    • A sintaxe básica de uma media query envolve o uso da regra @media, seguida de uma condição ou conjunto de condições. Quando essa condição é satisfeita, o conjunto de estilos dentro da media query é aplicado.

    Exemplo Simples: Vamos usar uma media query para alterar o layout de uma página quando a largura da janela for menor que 600 pixels (comum em dispositivos móveis).

    O padrão de desenvolvimento é começar pelo mobile primerio isso implica usar o (min-width ao inves de usar o max-width) a ordem de codar importa!

    Como Funciona:

    Estilos padrão:
    No exemplo acima, o corpo da página terá um fundo azul e um tamanho de fonte de 18px em qualquer tela que não corresponda à media query.

    Media query aplicada:
    Se a largura da janela do navegador for menor que 600px, o fundo muda para verde e o tamanho da fonte diminui para 16px.

    Tipos de Condições nas Media Queries:

    Largura e Altura da Tela (width, height):

    min-width: Aplica os estilos se a largura da tela for maior ou igual a um valor específico.

    max-width: Aplica os estilos se a largura da tela for menor ou igual a um valor específico.

    Orientação da Tela (orientation):

    landscape: Aplica os estilos quando a tela estiver em modo paisagem (horizontal).
    portrait: Aplica os estilos quando a tela estiver em modo retrato (vertical).

    Resolução de Tela (resolution):
    Aplica os estilos dependendo da densidade de pixels do dispositivo.

    Combinações de Condições: Você pode combinar várias condições usando operadores lógicos como and, not, e or.
