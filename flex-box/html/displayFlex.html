Flex-flow

é uma abreviação que combina duas outras propriedades do Flexbox, flex-direction e flex-wrap. Ela é usada para definir tanto a direção do fluxo de itens flexíveis quanto se os itens devem ser dispostos em múltiplas linhas.
sintaxe: flex-flow: [flex-direction] + [flex-wrap]; pode se usar:

  • Horizontal sem quebra - flex-flow: row nowrap;
  • Horizontal com quebra - flex-flow: row wrap;
  • Vertical sem quebra - flex-flow: column nowrap;
  • Vertical com quebra - flex-flow: column wrap;

flex-flow: row nowrap;

Os itens serão dispostos em uma única linha, da esquerda para a direita, sem quebra.
